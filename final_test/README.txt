В этой папке хранятся файлы, в которых я написал консольную игру "Крестики-нолики"

main.py - основная логика хода, главный запускаемый файл
bot_side.py - функции действия бота, при определенной стороне(крестик или нолик)
consts.py - константы примененные в игре, словари
decorators.py - декоратор
funcs.py - функции примененные в main

path для корректного функционирования зависимостей:
path/to/main> python main.py
